# Main-page-Houzzy
Главная страница и бургерное меню для сайта интернет-магазина Houzzy

# Описание проекта

## Техническое задание

Этот проект выполнен в соответствии со следующими техническими требованиями:

*   **Адаптивность и резиновая вёрстка**:
    *   Применена резиновая вёрстка для корректного отображения страницы на любой ширине от 360 до 4 000 пикселей.
    *   Вёрстка адаптирована под все разрешения, указанные в макете.
*   **Оптимизация изображений**:
    *   Растровые изображения подготовлены для экранов с плотностью пикселей 2х.
    *   Изображения подключены с учётом их адаптации.
    *   Векторные изображения добавлены в спрайт.
    *   Графика выбрана в подходящем формате: JPG, PNG или SVG.
*   **Стилизация и препроцессинг**:
    *   Стилизация выполнена с помощью препроцессора Sass.
    *   Семейства шрифтов и цвета записаны в SCSS-переменных.
    *   Сформированы все состояния элементов из UI-kit.
*   **Структура и семантика**:
    *   Вёрстка создана с помощью БЭМ (Блочно-элементная методология).
    *   Разметка сформирована согласно семантике HTML5.
    *   Основное содержимое страницы не должно находиться вне разделов. Весь контент внутри тега `<main>` располагается в тегах `<section>`.
    *   При формировании разделов горизонтальная центровка относительно краёв страницы выполняется за счёт утилитарного класса `<div class=”container”>`.
*   **Формы**:
    *   Поля формы находятся внутри тега `<form>`.
    *   У обязательных полей есть атрибут `required`.
    *   У полей формы указаны атрибуты `name` и `id`, у радиокнопок указан `value`.
    *   У кнопки верно заполнен обязательный атрибут `type`.
*   **Интерактивность и доступность**:
    *   Область взаимодействия с интерактивным элементом предсказуема и достаточна.
    *   У интерактивных элементов при фокусе с клавиатуры есть визуальный отклик.
    *   У всех интерактивных элементов есть явная или скрытая текстовая подпись.
    *   Добавлен скрытый заголовок при отсутствии подходящего заголовка из макета.
*   **Производительность и оптимизация**:
    *   У спозиционированных графических элементов отключена кликабельность (`pointer-events: none;`).
    *   Удалось избежать избыточного увеличения веса селектора.
*   **Соответствие макету и типографика**:
    *   Вёрстка соответствует макету, учитывая проверку по Pixel Perfect (погрешность — 10 px).
    *   При переносе из макета текст прошёл обработку типографом.
*   **Валидация и структура проекта**:
    *   Соблюдена логика файловой структуры проекта.
    *   Код отформатирован.
    *   Вёрстка прошла проверку на валидность (HTML-валидатор, БЭМ-валидатор).
*   **Обработка контента**:
    *   Вёрстка проверена на переполнение контентом.
    *   При недостаточной высоте основного содержимого подвал страницы прижат к низу экрана.
*   **Ссылки**:
    *   Всем ссылкам задано значение `#` для атрибута `href`.
